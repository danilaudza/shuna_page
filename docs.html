Shuna
Author Documentation

Pengetahuan Umum Shuna!

Gunakan /help untuk mencari tahu command dan website Shuna!

Prefix bawaan shuna adalah "+"

Welcoming Message

Pakai slash command /welcome

/welcome [channel] [message]

Contoh =>

/welcome #welcome-channel Selamat datang di discord kami @

Command Description

[channel]

 Pilih channel untuk dijadikan tempat notifikasi

[message]

 Isi pesan ketika guild member masuk

@

 Akan berisi user yang masuk

Prefix Command

Macam command yang menggunakan Prefix

Command Description

+ping

 Mengecek apakah Shuna bangun

Slash Command

Macam Slash command

Command Description

/welcome

 Memilih channel dan pesan untuk menyambut member baru

/help

 Mencari pertolongan tentang Shuna

/setnick

 Mengganti nickname daru member

/newprefix

 Mengganti prefix defautl maupun yang sudah di custom

/support

 Memberikan support pada Shuna melalui pesan, saran, tanggapan, laporan bug, dll

/ban

 Ban member nakal

/kick

 kick member nakal

/unban

 Unban member yang telah di ban

Music Bot

Command Description

Play music

+play [judul lagu]

 Mainkan musik dari youtube/spotify/soundcloud

Skip music

+skip

 Skip musik dari playlist

Stop music

+stop

 Hentikan Shuna bermain musik

Cek playlist

+queue

 Cek playlist dari musik yang dimainkan

Toggle autoplay

+autoplay

 Toggle autoplay on/off [default: on]

Toggle repeat

+repeat

 Toggle repeat lagu/playlist/off [default: off]
